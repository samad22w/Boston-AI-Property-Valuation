#  Boston AI Intelligence - Next-Gen Property Valuation
An advanced Machine Learning web application that predicts property values in Boston with an interactive UI, PDF reporting, and Voice features.

##  Key Features
- **XGBoost Regressor:** High-accuracy prediction model.
- **Dynamic PDF Reports:** Generates professional valuation reports for clients.
- **Voice Feedback:** Integrated gTTS for audio prediction.
- **Interactive UI:** Built with Streamlit, featuring Dark/Light mode and global property comparison.
- **Currency Converter:** Real-time valuation in USD, PKR, and INR.

##  Tech Stack
- **AI/ML:** XGBoost, Scikit-learn
- **Frontend:** Streamlit, CSS (Custom Theming)
- **Utilities:** FPDF (PDF Gen), gTTS (Voice), Pandas, NumPy

##  Setup
1. Clone the repo.
2. Install requirements: `pip install -r requirements.txt`
3. Run: `streamlit run app.py` (Access Key: admin)
